# Career Counseling Agent - HW11

This project is a Career Counseling Agent built with Ollama (llama3.2), LangChain, and Streamlit. 

## Features

- Skills Gap Analyzer: Compares user skills against job requirements and suggests learning paths.
- Resume Scorer: Evaluates resumes and provides actionable feedback and scoring.
- Salary Estimator: Provides salary ranges based on job title, location, and experience.
- Interview Question Generator: Produces technical and behavioral interview questions tailored to roles and difficulty levels.

## Usage

Run the agent locally using Streamlit:

```bash
pip install -r requirements.txt
streamlit run career_counseling_agent.py
```

Enter your queries in the chat interface to interact with the agent.

## Notes

- Conversation memory maintains context across messages.
- Tools are automatically called based on user input.
